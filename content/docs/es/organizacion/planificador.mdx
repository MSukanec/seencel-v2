# Planificador

El **Planificador** de SEENCEL es tu centro de planificación. Desde acá organizás todo lo que necesita fecha, seguimiento o coordinación en tu organización y proyectos.

El Planificador ofrece **tres modos de visualización** sobre el mismo conjunto de datos, accesibles desde el selector de la barra de herramientas:

| Modo | Para qué sirve | Ideal para |
|------|----------------|------------|
| **Lista** | Ver eventos en orden cronológico | Visión rápida de qué viene próximamente |
| **Panel** | Organizar tareas por estado (columnas) | To-dos, coordinación, seguimiento |
| **Calendario** | Cuadrícula mensual con eventos | Reuniones, plazos, hitos |

---

## ¿Dónde lo encuentro?

El Planificador está disponible en dos lugares:

- **A nivel de organización**: En el menú lateral, bajo "Planificador". Muestra todos los eventos y tareas de toda la organización.
- **A nivel de proyecto**: Dentro de cada proyecto, bajo "Planificador". Muestra solo lo asociado a ese proyecto.

<Callout type="tip" title="Filtrado automático">
Cuando accedés al Planificador desde un proyecto, los eventos y tarjetas se filtran automáticamente para mostrar solo lo relevante a ese proyecto.
</Callout>

## Eventos

Los eventos son items con fecha y hora: reuniones, hitos, plazos de entrega.

### Crear un Evento

1. Hacé click en el botón **"Nuevo Evento"** en la barra de herramientas, o
2. Hacé click directamente en un **día del calendario** (se pre-selecciona esa fecha)

### Campos del Evento

| Campo | Descripción | Obligatorio |
|-------|-------------|:-----------:|
| **Título** | Nombre descriptivo del evento | ✅ |
| **Proyecto** | Asociar a un proyecto específico (o dejar "Global") | No |
| **Todo el día** | Si el evento no tiene hora específica | No |
| **Fecha inicio** | Cuándo comienza el evento | ✅ |
| **Hora inicio** | Hora de inicio (si no es todo el día) | No |
| **Fecha fin** | Cuándo termina el evento | No |
| **Hora fin** | Hora de fin (si no es todo el día) | No |
| **Ubicación** | Lugar del evento (ej: "Oficina", "Obra Norte") | No |
| **Color** | Color visual para diferenciar tipos de eventos | No |
| **Descripción** | Notas adicionales sobre el evento | No |

### Colores de Eventos

Podés asignar un color a cada evento para identificarlos visualmente:

| Color | Nombre |
|-------|--------|
| 🔵 | Azul (predeterminado) |
| 🔴 | Rojo |
| 🟢 | Verde |
| 🟡 | Amarillo |
| 🟣 | Púrpura |
| 🟠 | Naranja |
| ⚫ | Gris |

## Panel de Tareas

El **Panel de Tareas** es tu espacio para organizar ideas, pendientes y cosas por hacer. Usá columnas que representen estados y arrastrá tarjetas entre ellas.

<Callout type="info" title="¿Para qué sirve?">
El panel de tareas está pensado para gestión general: ideas, pendientes, tareas administrativas, coordinación de equipo. Para tareas de construcción específicas de una obra (hormigonado, mampostería, etc.) usá las **Tareas de Obra** dentro de cada proyecto.
</Callout>

### Columnas

Cada panel tiene columnas que representan estados. Podés agregar, reordenar y renombrar columnas:

| Flujo simple | Flujo de gestión | Flujo de ideas |
|-------------|-----------------|----------------|
| Por hacer | Recibido | Idea |
| En progreso | En revisión | En evaluación |
| Terminado | Aprobado | Aprobada |
| | Archivado | Descartada |

### Tarjetas

Hacé click en **"+ Agregar tarjeta"** dentro de una columna para crear una tarea.

| Campo | Descripción | Obligatorio |
|-------|-------------|:-----------:|
| **Título** | Nombre de la tarea o idea | ✅ |
| **Portada** | Imagen de portada visual | No |
| **Proyecto** | Asociar a un proyecto | No |
| **Asignado a** | Miembro responsable | No |
| **Prioridad** | Urgente, Alta, Media, Baja o Sin prioridad | No |
| **Fecha inicio** | Cuándo empieza | No |
| **Fecha límite** | Cuándo debe estar lista | No |

### Prioridades

| Prioridad | Indicador |
|-----------|-----------|
| **Urgente** | 🔴 Rojo |
| **Alta** | 🟠 Naranja |
| **Media** | 🟡 Amarillo |
| **Baja** | 🔵 Azul |
| **Sin prioridad** | Sin color |

### Arrastrar y Soltar

- **Mover tarjetas** entre columnas para cambiar su estado
- **Reordenar tarjetas** dentro de una columna para definir prioridad visual
- **Reordenar columnas** arrastrándolas por su encabezado

## Filtros

Cada modo ofrece filtros específicos en la barra de herramientas:

| Modo | Filtros disponibles |
|------|---------------------|
| **Lista / Calendario** | Tipo (Manual, Kanban, Pagos, Hitos) |
| **Panel** | Prioridad, Etiquetas |

Además, la **búsqueda por texto** está disponible en todos los modos para filtrar por título y descripción.

---

## Navegación del Calendario

- Usá las flechas **← →** para moverte entre meses
- Hacé click en **"Hoy"** para volver al mes actual
- En la vista de lista, los eventos se agrupan por fecha con etiquetas como "Hoy", "Mañana" o la fecha completa
